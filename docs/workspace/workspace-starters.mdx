---
id: workspace-starters
title: Workspace Starters
---

Generate a new workspace starter project with demo components.

```bash
bit new react my-workspace
```

Generate an empty workspace.

```bash
bit new react my-workspace --empty
```

Generate a workspace without git.

```bash
bit new react my-workspace --no-git
```

Generate a new React app with demo components.

```bash
bit new react-app my-app
```

Generates a new React lib with demo components.

```bash
bit new react-lib my-lib
```
