Bit helps developers compose and collaborate on modern applications with [independent components](/components/overview) and dependencies.
It makes development of component-driven and modular systems fast, standard, consistent and scalable.

## Installing Bit

Install Bit using BVM, Bit's version manager.

<Tabs
defaultValue="NPM"
values={[
{label: 'NPM', value: 'NPM'},
{label: 'Yarn', value: 'Yarn'},
]}>
<TabItem value="NPM">

```bash
npm i -g @teambit/bvm
```

  </TabItem>
  <TabItem value="Yarn">

```bash
yarn global add @teambit/bvm
```

  </TabItem>
</Tabs>

```bash
bvm install
```

## BVM and Bit Versions

To check your version of BVM and Bit and see what the latest version is run the bvm version command.

```js
bvm version
```

### Upgrading Bit

To upgrade to the latest version of Bit:

```bash
bvm upgrade
```

## Manage versions

Get version information for Bit and BVM.

```bash
bvm version
```

List available versions of Bit and BVM.

```bash
bvm list --remote
```

List installed versions

```bash
bvm list
```

Remove a local version of Bit so it will fallback to a previous version.

```bash
bvm remove <bit-version>
```

### Installing with a CommonJS package manager

Bit can be installed with a CommonJS package manager.
Please note, it is not recommended to install Bit directly from NPM. We recommend on using [BVM](/reference/bvm).
