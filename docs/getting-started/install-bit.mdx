Install Bit using [BVM](/reference/using-bvm#install-bvm), a version manager for Bit. Using BVM makes it easier to install and manage multiple versions of Bit in a single environment.

## Installing Bit

Install Bit using BVM, Bit's version manager.

<Tabs
defaultValue="NPM"
values={[
{label: 'NPM', value: 'NPM'},
{label: 'Yarn', value: 'Yarn'},
]}>
<TabItem value="NPM">

```bash
npm i -g @teambit/bvm
bvm install
```

  </TabItem>
  <TabItem value="Yarn">

```bash
yarn global add @teambit/bvm
bvm install
```

  </TabItem>
</Tabs>

## BVM and Bit Versions

To check your version of BVM and Bit and see what the latest version is run the bvm version command.

```js
bvm version
```

### Upgrading BVM

To upgrade to the latest version of BVM:

### Upgrading Bit

To upgrade to the latest version of Bit:

```bash
bvm upgrade
```

## Troubleshooting

If you get a warning or the bvm command is not available please see the [BVM guide](/reference/using-bvm#install-bvm).
