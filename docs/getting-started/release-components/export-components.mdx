## Overview

Learn how to export your components to a remote scope so they are available for other developers to install and use.

## Exporting your Components

Once you have tagged your component with the `bit tag --all` command and set your default Scope in the `workspace.jsoc` then you can then export your components to a remote scope by running the following command:

```bash
bit export
```
